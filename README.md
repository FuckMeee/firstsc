Spring Cloud Demo
====================

#### 介绍
基于springboot和springcloud的服务夫基础框架搭建。  
仅供自己学习之用  

* auth 认证服务
* config 配置文件
* config-client spring-cloud-config 客户端
* config-server spring-cloud-config 服务端
* consumer 消费者
* eureka-discovery 服务注册中心
* gateway spring-cloud-gateway网关
* provider 服务提供者
* zuul zuul网关
